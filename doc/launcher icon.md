# Launcher Icon

## DALL·E

이미지 그릴 실력도, 시간도 없기 때문에 애플리케이션 아이콘을 [DALL·E](https://labs.openai.com)로 생성했다. 마침 공개로 풀렸으니까.

그런데 만들어보면 볼수록 어디서 본것 같은 결과만 나온다. 그냥 비슷한 검색어로 검색하면 거의 같은 이미지가 나오기 때문에 DALL·E로 아이콘 만들기는 포기.
